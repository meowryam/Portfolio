Project Overview

A nostalgic arcade game, reimagined.

A modern remake of the classic Snow Bros arcade game built with C++ and SFML. Players freeze enemies into snowballs and launch them to create chain reactions across tile-based levels.

Inspired by Disney's Frozen, the project features an icy interface, custom menus, sprite animations, and clean object-oriented architecture designed for maintainability and future expansion.
