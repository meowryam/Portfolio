Project Overview

Three arcade classics, one application.

Gaming Hub brings together three classic games Brick Breaker, Ping Pong, and Feed the Snake into a single C++ and SFML application. Each game runs independently with its own mechanics, but shares a cohesive pastel arcade style inspired by cute retro aesthetics.

Every detail, from the sakura-themed backgrounds to the heart-shaped UI elements, was designed to create a cohesive and playful arcade experience. The interface combines retro inspiration with a distinctive pastel aesthetic.
